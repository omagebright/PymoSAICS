# MOSAICS KB_3pt assets

These are the historical MOSAICS three-point protein/nucleic-acid topology and
knowledge-based parameter files supplied with the Oxford 7QPJ project from Tom
Ferdinandt on 2026-07-20. Redistribution is covered by the maintainer's
2026-07-21 authorization recorded in `../../ASSET_NOTICE.md`.

The parameter file is used for MOSAICS bond, bend, torsion, one-four, and
knowledge-based interaction directives. PymoSAICS does not reinterpret it.

PymoSAICS can construct a transparent initial CA/O/CMA model from a canonical
protein PDB. That conversion is a new coordinate preparation, not a claim to
reproduce a historical project's exact side-chain preparation, STRIDE
segmentation, hierarchical regions, or experimental cryo-EM observations.
